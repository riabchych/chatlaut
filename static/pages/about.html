Цели и задачи

При создании данного прооекта стояла задача — сделать наиболее удобное, быстрое и эффективное решение для группового общения из всех существующих.

Особое внимание было уделено именно удобству пользовательского интерфейса и скоростю работы.

Технические требования

  • Включенный JavaScript
  • Браузер поддерживающий стандарт HTML5

Используемые технологии

  • NodeJS
  • Google WebSocket
  • jQuery JS Library
  • Socket.IO
  • MongoDB
